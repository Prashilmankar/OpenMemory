# 🧠 OpenMemory

> **The open-source memory layer for AI applications.**

Store, search, and retrieve long-term memory for LLM-powered applications.

---

## ✨ Features

- 🧠 Long-term memory
- 🔍 Semantic search
- ⚡ FastAPI REST API
- 📦 SQLite metadata storage
- 🚀 FAISS vector search
- 🤖 Local embeddings
- 🔒 Privacy-first
- 🌍 Open source

---

## 🛠 Tech Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite
- FAISS
- Sentence Transformers
